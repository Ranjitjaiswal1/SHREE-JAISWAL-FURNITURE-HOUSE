# Shree Jaiswal Furniture House Website

A modern React website for Shree Jaiswal Furniture House located in Ghorasahan, East Champaran, Bihar.

## Features

- Home page with business highlights
- Products catalog (Beds, Sofas, Dining Tables, Wardrobes, Office Furniture)
- About Us page
- Gallery with furniture photos
- Contact page with inquiry form and Google Maps integration
- WhatsApp integration for inquiries

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
